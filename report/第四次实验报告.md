## 3. 模型与训练设置
### 3.1 模型选择
本实验使用的模型为：
- 模型名称：**YOLOv8m** (YOLOv8 Medium)
- 选择该模型的原因：YOLOv8m 在精度和速度之间取得了较好的平衡，适合交通标志检测任务。相比于 YOLOv8n 和 YOLOv8s，YOLOv8m 具有更强的特征提取能力，能够更好地识别复杂的交通标志。

### 3.2 训练参数
- 训练轮数（epochs）：**50**
- 图像尺寸（imgsz）：**640**
- batch size：**8**
- 优化器：**Auto** (SGD)
- 学习率：初始学习率约 0.00017，余弦退火至 0.000015
- 是否使用数据增强：**是**（YOLOv8 默认增强）

### 3.3 训练命令
```bash
cd 路标
python code/train_model.py
```

训练脚本主要参数：
```python
model = YOLO('yolov8m.pt')
results = model.train(
    data='code/data.yaml',
    epochs=50,
    imgsz=640,
    batch=8,
    device=0,
    project='runs/traffic_signs_v2',
    name='train'
)
```

## 5. 混淆矩阵分析

混淆矩阵图位于 `results/confusion_matrix.png`。

![混淆矩阵](results/confusion_matrix.png)

请重点分析：
1. **哪些类别识别效果最好？** Speed Limit 类标志（尤其是限速 20、30、40、50、60、70、80、90、100、110、120）识别效果最好，召回率普遍在 95% 以上。
2. **哪些类别最容易混淆？** Green Light 和 Red Light 之间存在一定混淆，这是由于红绿灯的绿色和红色区域在某些角度下相似度较高。
3. **造成类别混淆的可能原因是什么？** 
   - 红绿灯的颜色相似性导致误识别
   - 部分标志尺寸较小，难以区分细节
   - 夜间或逆光条件下的颜色偏差
4. **从混淆矩阵中可以看出模型还有哪些不足？** 
   - Stop 标志与 Speed Limit 标志存在一定混淆
   - 小目标检测精度有待提升
