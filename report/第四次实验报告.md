# 第四次实验报告 - 交通标志目标检测

## 实验目标

使用 YOLOv8 进行交通标志目标检测，预测测试集中的交通标志类别和位置。

## 数据集说明

- **训练集**: 包含带标注的训练图像
- **验证集**: 用于模型验证
- **测试集**: 638张图片，需要预测

### 类别列表 (15类)

| 类别ID | 名称 |
|--------|------|
| 0 | Green Light |
| 1 | Red Light |
| 2-13 | Speed Limit (10-120) |
| 14 | Stop |

## 实验方法

### 模型选择
- 使用 YOLOv8m (Medium) 预训练模型
- 相比 nano 版本，m 版本有更好的检测精度

### 训练参数
- Epochs: 50
- Image Size: 640
- Batch Size: 8
- 设备: GPU (RTX 4060)
- 数据增强: Mosaic, Mixup, Copy-Paste

## 实验结果

- **mAP@0.5**: 87%
- **Recall**: 81%
- **Precision**: 较高

## 结论

使用 YOLOv8m 模型配合数据增强策略，成功实现了交通标志的高精度检测，mAP 达到 87%。
